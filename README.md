# s-pipe-fsi-analysis
Two-way FSI analysis of an S-shaped pipe in ANSYS Workbench 2024 R1 — transient CFD (Fluent) ↔ structural FEA (Mechanical) via System Coupling, plus modal, harmonic response (DAF), and high-cycle fatigue (Goodman) assessment. Includes full report, exported data, and result contours.
